# Party Menu Planner App

A React application built with Vite that allows users to browse a 37-item party menu, view deep recipe details (including ingredients and instructions), and save favorite items to a persistent party planning list.

## Features
- **Recipe Catalog:** Browse recipes sorted by category.
- **Dynamic Recipe View:** Detailed view for all dishes with ingredients and step-by-step instructions.
- **Party Planner List:** Context-driven save functionality to bookmark recipes.
- **Responsive UI:** Clean dark-themed interface built with CSS.

## Setup Instructions
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local development server.
